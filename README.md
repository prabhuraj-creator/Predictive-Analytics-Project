# Predictive Analytics Using Historical Data

## Objective
This project predicts future sales trends using machine learning techniques.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm Used
Linear Regression

## Features
- Historical data analysis
- Data preprocessing
- Sales prediction
- Model evaluation
- Visualization

## Output
- R2 Score
- MAE Score
- Actual vs Predicted Graph

## Conclusion
Predictive analytics helps businesses forecast future sales and improve strategic planning.
